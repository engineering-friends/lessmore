

# About 
This is an alpha. Planning on making a public library.

Formats steps comments: 
- adds new lines before and after if needed
- adds title case 
- removes trailing dot 
- splits dashes by three (`----` -> `--- -`)


```
# - Step 1

a = 1
b = 2

c = 3

# - Step 2

a = 2

# - Step 3

# -- Step 3.a

a = 1

# -- Step 3.b

b = 2

# - Step 4

# -- Step 4.1

# --- Step 4.1.1

# --- - Step 1.1.1.1

# --- -- Step 1.1.1.1.1

# --- --- Step 1.1.1.1.1.1.

# --- --- - Step 1.1.1.1.1.1
```
